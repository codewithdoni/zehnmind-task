## Getting started

```bash
# 1. Install deps
flutter pub get

# 2. Generate translations
dart run slang

# 3. Generate freezed/injectable code
dart run build_runner build --delete-conflicting-outputs

# 4. iOS only
cd ios && pod install && cd ..

# 5. Run
flutter run
```

## Firebase setup

The Firebase project (`zehnmind-task-9f2a3`) is already configured:
- `lib/firebase_options.dart` (generated by `flutterfire configure`)
- `android/app/google-services.json`
- `ios/Runner/GoogleService-Info.plist`

In Firebase Console enable:
- **Authentication** → Email/Password, Phone, Google providers
- **Firestore Database** → Native mode
- **Crashlytics** + **Analytics**

Deploy security rules:
```bash
firebase deploy --only firestore:rules
```

For Google Sign-In on Android, add SHA-1 fingerprint to Firebase Console:
```bash
cd android && ./gradlew signingReport
```

For iOS Phone Auth, upload APNs key to Firebase Console (Project Settings → Cloud Messaging).

## Code style highlights

- **Clean Architecture**: each feature has `data/`, `domain/`, `presentation/` layers
- **Repositories**: abstract `BaseXxxRepository` in `domain/`, concrete `@Injectable(as: BaseXxxRepository)` in `data/`
- **Data sources**: `@lazySingleton` Firestore wrappers (`AuthFirebaseSource`, `TodoFirestoreSource`, `ProfileFirestoreSource`)
- **BLoC**: `@injectable` with freezed sealed events/states (status enum + `error` field)
- **Error handling**: all Firebase exceptions mapped to `AppException` subclasses through `SafeExecutionManager.run()`
- **Reactive**: `emit.forEach` for Firestore streams (TodoBloc.startWatching, ProfileBloc.load)

## Firestore data model

```
users/{uid}
  ├── full_name: String
  ├── email: String
  ├── phone: String
  ├── created_at: Timestamp
  └── todos/{todoId}
      ├── title: String
      ├── description: String
      ├── is_completed: bool
      ├── created_at: Timestamp
      └── updated_at: Timestamp
```

Security rules (`firestore.rules`) restrict access to the authenticated user's own data.
